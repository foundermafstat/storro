# ChatGPT App Integration

Storro exposes the same authenticated MCP tools to the ChatGPT App integration.

The app can:

- Connect a Storro user through OAuth.
- Let the user choose a Storro project.
- Send only user-selected notes or excerpts into Storro as source records.
- Queue story generation from explicit Storro context.
- Retrieve Storro artifacts.
- Save draft revisions back to Storro.

Privacy boundary: the ChatGPT App sends only user-selected context to Storro. It does not access all ChatGPT history or hidden conversations.
