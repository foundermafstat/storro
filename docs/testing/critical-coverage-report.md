# Critical Coverage Report

Generated by `npm run coverage:critical`.

## Critical Services

- `authorization-service`: authorization matrix and admin/billing boundaries.
- `billing-service`: Checkout, Stripe webhook replay, quota enforcement.
- `security-service`: encryption, rate limiting, data export/deletion.
- `github-webhook-service`: signature validation and replay/idempotency.
- `ai-gateway`: retries, schema parsing, usage accounting.
- `extraction-pipeline-service`: extraction jobs and fact persistence.
- `grounding-review-service`: hallucination, secret, and generic phrase blocking.
- `artifact-generation-service`: approved plan to artifact.
- `artifact-export-service`: markdown/export generation and override audit.
- `notification-service`: preferences, critical notifications, dedupe.
- `observability-service`: Sentry reporting, metrics, job traces.

## Launch Gate

CI must pass:

- `npm run test:unit`
- `npm run test:integration`
- `npm run test:e2e`
- `npm run test:ai-eval`
- `npm run coverage:critical`
